C:\Users\<user>\AppData\Local\GitHub\PortableGit_<guid>\cmd
#+END_EXAMPLE

- about:config -> browser.startup.Homepage -> about:newtab

** Disable youtube fullscreen notification
- about:config -> full-screen.api.warning.timeout -> 0
